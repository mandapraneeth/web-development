Practice Exercises

  1. Create a box with 30px padding and 50px margin.
  2. Make a flex container with 3 items aligned horizontally and spaced evenly.
  3. Create a column layout using flex-direction: column.
  4. Vertically center a box inside a container using align-items: center.

Solutions

  1. 
.box {
  padding: 30px;
  margin: 50px;
  border: 1px solid black;
}
        
  2. 
.container {
  display: flex;
  justify-content: space-evenly;
}
        
  3. 
.flex-column {
  display: flex;
  flex-direction: column;
}
        
  4. 
.align-center {
  display: flex;
  align-items: center;
  height: 100px;
}
