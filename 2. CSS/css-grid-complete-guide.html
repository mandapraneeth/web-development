Assignment: Grid-Based Product Card Layout

Task: Create a product grid with 3 columns that turns into 1 column on small screens.

.product-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}
@media (max-width: 768px) {
  .product-grid {
    grid-template-columns: 1fr;
  }
}

Solution Tip: Use media queries to adjust grid-template-columns.
